Notes

  • Access Modifier: allows us to define scope of the code, that is how one part of the same or other code can access this code. Ex- public (gives full access)
  • Void: return type that shows that the method won't return anything
  • Parameters: ways to pass information to a method
  • String Literal: enclosed within double quotes. value can't be changed.
  • Operators: perform an operation on a variable or value
  • Primitive Data Types: 8 types. There's a wrapper class(here, Type) for all the 8 types (Type.MAX_VALUE;).
    • Overflow- when you assign more than max range of a data type, it goes to the min value.
    • Underflow- when you assign less than the min value of a data type, it goes to the max value.
    • Width- Occupies 'x' bits= has a width of 'x'.
